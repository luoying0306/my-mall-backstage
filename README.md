## 启动redis说明 windows版本：
1. 进入redis目录看到redis-server.exe目录
2. 在地址栏输入cmd 回车
3. 输入 `redis-server.exe redis.windows.conf` 回车
4. 看到控制台有redis的图标，即启动成功
